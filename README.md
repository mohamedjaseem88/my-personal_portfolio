A modern, animated developer portfolio with smooth transitions, glassmorphism UI, and interactive 3D elements.

---

## ✨ Features

- Dark modern UI with animations
- Smooth scroll effects
- 3D visuals
- Fully responsive (mobile + desktop)
- Contact form connected to Firebase
- AI chatbot using Google AI Studio
- Social links and clean layout

---

## 🛠️ Tech Stack

- React + Tailwind CSS
- Framer Motion
- Three.js
- Firebase (Database)
- Vite

---

## 🔑 Environment Setup

Create a file named **.env.local** in the root folder and add:

```env
VITE_FIREBASE_API_KEY=your_key
VITE_FIREBASE_AUTH_DOMAIN=your_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
GEMINI_API_KEY=your_gemini_key
▶️ Run Project
npm install
npm run dev

Open in browser:
http://localhost:5173/

🗄️ Firebase Setup (Contact Form)
Create project in Firebase
Enable Firestore Database
Create collection: contact_messages
